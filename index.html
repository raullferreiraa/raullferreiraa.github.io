Projeto em Destaque

GitHub

Garagem Digital

Plataforma automotiva Full Stack com autenticação, feed de projetos, garagem pessoal e recursos sociais.

Projeto desenvolvido com Python, Flask, MySQL, HTML, CSS e JavaScript Vanilla. A aplicação começou como um CRUD de veículos e evoluiu para a base inicial de uma plataforma social automotiva, com autenticação, permissões por usuário, upload de imagens, curtidas, comentários, perfis simples e visualização detalhada dos projetos.

Destaques do Sistema

  • Cadastro e login de usuários
  • Feed geral e garagem pessoal
  • Controle de propriedade por usuário
  • Curtidas e comentários
  • Perfil simples de usuário
  • Modal com visualização detalhada
  • Upload de imagens
  • API REST com Flask e MySQL
